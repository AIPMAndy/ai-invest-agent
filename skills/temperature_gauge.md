# Skill: 指数温度计 (invest-temp)

## 触发命令
```
/invest-temp
```

## Prompt模板

```
请执行指数温度计更新：

## 执行步骤

### 1. 数据获取
对以下33个指数，联网搜索获取：
- 当日收盘价
- 当日涨跌幅
- 今年以来涨跌幅
- PE-TTM
- PE历史分位（2021年至今）

### 2. 温度计算
```
PE温度 = PE历史分位 × 100
```

### 3. 市场整体温度
```
A股温度 = 所有A股指数PE温度的平均值
港股温度 = 所有港股指数PE温度的平均值
美股温度 = 所有美股指数PE温度的平均值
```

## 输出格式

### 表格格式（每个市场）
| 序号 | 板块 | 指数名称 | 当日 | 今年以来 | PE(TTM) | PE温度 | 指数代码 |
|------|------|----------|------|----------|---------|--------|----------|

按当日涨跌幅从高到低排序

### 温度分布汇总
| 温度区间 | 估值状态 | 操作建议 | 对应指数 |
|----------|----------|----------|----------|
| 0-20℃ | 严重低估 | 重仓买入 | xxx |
| 20-30℃ | 低估 | 分批买入 | xxx |
| 30-50℃ | 合理 | 持有观望 | xxx |
| 50-70℃ | 合理偏高 | 谨慎持有 | xxx |
| 70-90℃ | 高估 | 分批卖出 | xxx |
| 90-100℃ | 严重高估 | 清仓卖出 | xxx |

### 本期操作计划
1. 市场整体估值判断
2. 低估可买入板块（温度≤30℃）
3. 高估需卖出板块（温度≥70℃）
4. 本期持仓调整动作
```
