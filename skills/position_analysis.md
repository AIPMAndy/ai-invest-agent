# Skill: 标的深度分析 (invest-position)

## 触发命令
```
/invest-position
```

## Prompt模板

```
请执行标的深度分析：

## 输入
读取用户提供的持仓表格，包含：
- 板块
- 持仓比例
- 标的名称
- 成本价
- 底仓设定
- 优先级

## 执行步骤

### 1. 数据获取
对每个持仓标的，联网搜索：

**财务数据**:
- "{股票名称} {年份} 业绩 营收 净利润"
- "{股票名称} 毛利率 ROE 现金流"

**估值数据**:
- "{股票名称} PE估值 PB 历史分位"
- "{股票名称} 机构目标价 研报"

**行业数据**:
- "{股票名称} 市场份额 竞争格局"
- "{股票名称} 护城河 竞争优势"

**趋势数据**:
- "{股票名称} 股价走势 技术分析"

### 2. 10维度分析框架

对每个标的，输出以下10个维度的分析：

#### ① 基本面核心数据
- 最新财务数据（营收/净利/同比增速）
- 毛利率/净利率/ROE
- 经营现金流
- 业务结构与占比

#### ② 估值多维度分析
- 当前PE/PB
- 历史分位（近5年）
- 机构目标价
- 同业估值对比
- 估值空间判断

#### ③ 行业地位与护城河
- 行业地位描述
- 护城河因素（✓优势 / ✗劣势）
  - 品牌
  - 技术
  - 规模
  - 网络效应
  - 成本优势
  - 转换成本

#### ④ 核心驱动因素
列出3-5个可能推动股价上涨的因素

#### ⑤ 核心风险因素
列出3-5个可能导致股价下跌的因素
标注风险等级：低/中/高

#### ⑥ 趋势与买卖点
- 周线/日线趋势判断
- 关键支撑位
- 关键压力位
- 建议买入区间
- 建议卖出区间

#### ⑦ 长期温度
基于长期估值判断：
严重低估 / 低估 / 合理 / 高估 / 严重高估

#### ⑧ 短期温度
基于短期趋势判断：
严重低估 / 低估 / 合理 / 高估 / 严重高估

#### ⑨ 综合评级
五星制评级：★☆☆☆☆ 到 ★★★★★

#### ⑩ 操作建议
- 当前判断（1句话）
- 具体操作建议（2-3条）
- 注意事项

---

## 持仓健康度评估

### 板块配置评估
| 板块 | 当前仓位 | 目标仓位 | 偏离度 | 调整建议 |
|------|----------|----------|--------|----------|

### 估值分布评估
| 估值状态 | 标的数量 | 仓位占比 | 健康度 |
|----------|----------|----------|--------|

### 操作清单
1. 需要减仓的标的
2. 需要加仓的标的
3. 需要清仓的标的
4. 待建仓标的

---

## 输出格式

输出为Excel表格，包含：
- 所有持仓标的的10维度分析
- 持仓健康度评估
- 操作清单

## 输出文件
- {日期}_标的深度分析.xlsx
```
