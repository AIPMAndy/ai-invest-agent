# AI投资Agent - 跨市场价值投资复盘系统

## 系统概述

基于四步法的AI投资复盘系统，覆盖A股/港股/美股三大市场，输出标准化的投资分析报告。

**核心理念**：30度之下买入，50度之上卖出

**更新频率**：每周五收盘后（价值投资风格）

## 核心交易规则

| 温度区间 | 估值状态 | 操作建议 | 仓位指导 |
|----------|----------|----------|----------|
| 0-20℃ | 严重低估 | 重仓买入 | 可加至满仓 |
| 20-30℃ | 低估 | 分批买入 | 逐步加仓 |
| 30-50℃ | 合理 | 持有观望 | 维持仓位 |
| 50-70℃ | 合理偏高 | 谨慎持有 | 停止加仓 |
| 70-90℃ | 高估 | 分批卖出 | 逐步减仓 |
| 90-100℃ | 严重高估 | 清仓卖出 | 清仓离场 |

## 工具使用

### MD转DOCX工具

```bash
# 安装依赖
pip install python-docx

# 使用方法
python tools/md2docx.py 报告.md 报告.docx 楷体
```

### Excel生成工具

```bash
# 安装依赖
pip install openpyxl

# 在Python中使用
from tools.create_excel import create_position_analysis_excel, create_temperature_excel

# 生成标的分析Excel
create_position_analysis_excel(data, "标的深度分析.xlsx")

# 生成温度计Excel
create_temperature_excel(data, "指数温度计.xlsx")
```
