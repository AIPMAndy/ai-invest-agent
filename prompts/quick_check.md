# 每日快速检查 - Prompt模板

## 使用方式

将以下Prompt复制到Claude Code中执行：

---

```
# AI投资Agent - 每日快速检查

今天是{当前日期}，请进行快速市场检查。

## 执行步骤

### 1. 三大市场行情
搜索并汇总：
- A股：上证指数、沪深300、创业板指 今日涨跌幅
- 港股：恒生指数、恒生科技 今日涨跌幅
- 美股：标普500、纳斯达克（昨收）涨跌幅

### 2. 资金流向
- 北向资金：今日净流入/流出
- 南向资金：今日净流入/流出
- 两融余额变化

### 3. 重大新闻
搜索最近24小时内：
- 宏观政策变化
- 行业重大事件
- 公司重要公告

### 4. 持仓标的表现
根据我的持仓配置，检查：
- 涨幅前3的标的
- 跌幅前3的标的
- 异动标的（涨跌幅>5%）

## 输出要求

只输出关键结论，不超过500字，格式如下：

---

## {日期} 市场快报

**行情**：A股xxx，港股xxx，美股xxx

**资金**：北向xxx亿，南向xxx亿

**要闻**：
1. xxx
2. xxx
3. xxx

**持仓**：
- 领涨：xxx
- 领跌：xxx
- 关注：xxx

**结论**：一句话总结今日市场

---
```

---

## 变体：盘前检查

```
# AI投资Agent - 盘前检查

今天是{当前日期}，请进行盘前快速检查。

搜索并汇总：
1. 美股隔夜表现（三大指数）
2. A50期货走势
3. 港股夜盘/ADR表现
4. 隔夜重大新闻

输出：今日A股/港股可能的开盘预期，不超过200字。
```

---

## 变体：盘后复盘

```
# AI投资Agent - 盘后复盘

今天是{当前日期}，请进行盘后快速复盘。

搜索并汇总：
1. 今日三大市场收盘数据
2. 板块涨跌排行（前5/后5）
3. 北向南向资金全天流向
4. 涨停跌停统计
5. 盘后重要公告

输出：今日市场总结+明日关注点，不超过300字。
```
