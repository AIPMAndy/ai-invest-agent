# 周度投资复盘 - 主Prompt

## 使用方式

将以下Prompt复制到Claude Code中执行：

---

```
# AI投资Agent - 周度复盘任务

今天是{当前日期}，请执行完整的四步投资复盘流程。

## 执行要求

1. **联网搜索**：必须联网获取最新数据，不使用过时信息
2. **客观分析**：基于数据给出客观判断，不盲目乐观或悲观
3. **输出格式**：同时输出MD和DOCX格式，DOCX使用楷体字体
4. **表格转换**：MD表格需正确转换为Word表格

## Step 1: 宏观分析

搜索并分析：
- 政策面：美联储/中国货币财政政策/地缘风险
- 资金面：美债/美元/中美利差/北向南向/黄金/比特币
- 基本面：美中经济数据+三大市场估值
- 重大事件与风险

输出：
- {日期}_宏观分析报告_详细版.md/.docx
- {日期}_宏观分析报告_简版.md/.docx

## Step 2: 板块分析

分析四大板块：科技/医疗/红利/消费金融
每个板块覆盖：A股/港股/美股

输出：
- {日期}_板块分析报告_详细版.md/.docx
- {日期}_板块分析报告_简版.md/.docx

## Step 3: 指数温度计

计算33个指数的PE温度：
- A股20个指数
- 港股8个指数
- 美股5个指数

温度 = PE历史分位 × 100
规则：30度之下买入，50度之上卖出

输出：
- {日期}_指数温度计.md/.docx

## Step 4: 标的深度分析

读取我的持仓数据，对每个标的进行10维度深度分析：
①基本面 ②估值 ③护城河 ④驱动因素 ⑤风险因素
⑥趋势买卖点 ⑦长期温度 ⑧短期温度 ⑨评级 ⑩操作建议

输出：
- {日期}_标的深度分析.xlsx

## 我的持仓数据

[在此粘贴持仓表格]

## 输出目录

所有文件保存到：./output/{日期}/

## 执行确认

完成后输出执行摘要，包括：
1. 三大市场整体温度
2. 最值得关注的3个机会
3. 最需要警惕的3个风险
4. 本周核心操作建议
```

## 变体Prompt

### 快速检查（每日）

```
# AI投资Agent - 每日快速检查

今天是{当前日期}，请进行快速市场检查。

搜索并汇总：
1. 三大市场今日涨跌幅
2. 北向/南向资金流向
3. 重大新闻事件
4. 我的持仓标的今日表现

只输出关键结论，不超过500字。
```

### 单标的分析

```
# AI投资Agent - 单标的深度分析

请对{股票名称}进行深度分析。

搜索最新数据，按10维度框架输出：
①基本面 ②估值 ③护城河 ④驱动 ⑤风险
⑥趋势 ⑦长期温度 ⑧短期温度 ⑨评级 ⑩建议

输出为结构化文本。
```

### 温度计快速更新

```
# AI投资Agent - 温度计快速更新

请更新以下指数的PE温度：
{指数列表}

输出格式：
| 指数 | PE | 温度 | 操作建议 |
```
